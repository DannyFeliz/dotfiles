# Setup
##### 1. Install zsh
 - Mac
   * `brew install zsh zsh-completions`
 - Linux
   * Centos: `sudo yum -y install zsh`
   * Ubuntu: `sudo apt-get install zsh`

##### 2. Install oh-my-zsh
- `sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"`

##### 3. Set as default
 * `chsh -s $(which zsh)`
 
##### 4. Exit and Enter :)
